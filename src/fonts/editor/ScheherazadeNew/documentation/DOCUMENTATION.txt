DOCUMENTATION
Scheherazade New
========================

This file describes the documentation files included with the Scheherazade 
New font. This information should be distributed along with the 
Scheherazade New font and any derivative works.

Primary documentation for the Scheherazade New font is on the Scheherazade 
website (http://software.sil.org/scheherazade/).

The ScheherazadeNew-features.pdf file illustrates the use of OpenType and 
Graphite features.

The ScheherazadeNew-features.odt file illustrates the use of OpenType and 
Graphite features (source file for the pdf).

The ScheherazadeNew-typesample.pdf file gives a sample of all the characters in 
the font.

The ScheherazadeNew-typesample.odt file gives a sample of all the characters in 
the font (source file for the pdf).
